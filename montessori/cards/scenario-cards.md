# 蒙氏家庭场景卡

## 1. 孩子不收玩具

结论：
先不要判断孩子不听话，先看玩具是否太多、归位是否清楚、收拾任务是否过大。

蒙氏视角：
孩子需要外部秩序来建立内部秩序。如果玩具都堆在一个大箱子里，孩子很难理解「每个东西回到哪里」。

家庭做法：
1. 同时只开放 3-5 个玩具。
2. 每个玩具有固定位置。
3. 用托盘、篮子、图片帮助孩子归位。
4. 成人先示范「拿出—使用—放回」。

避免做法：
- 一边骂一边替孩子收。
- 玩具全部混在一个大箱子里。
- 要求孩子一次收完整个房间。

观察任务：
接下来 3 天，只开放少量玩具，看孩子是否更容易放回原处。

## 2. 孩子反复倒水

结论：
如果安全，反复倒水可以被看成手眼协调、动作控制和秩序感练习。

蒙氏视角：
孩子不是在「玩水捣乱」，可能是在通过重复练习控制动作。

家庭做法：
1. 准备小水壶、小杯子、托盘和抹布。
2. 每次只放少量水。
3. 允许孩子重复倒。
4. 洒出来时，平静示范擦干。

避免做法：
- 直接说「别玩水」。
- 一洒就责备。
- 因为怕麻烦完全禁止。

观察任务：
给孩子一个安全倒水区，观察他重复几次后动作是否更稳定。

---

## 3. 孩子不愿意穿鞋

结论：
先看鞋子难度、成人节奏和出门时间，不要先把它归因成磨蹭。

蒙氏视角：
穿鞋是复杂动作，包括找鞋、分左右、伸脚、拉好。孩子可能只是其中某一步还不会。

家庭做法：
1. 选择容易穿脱的鞋。
2. 把动作拆成「拿鞋—分左右—脚伸进去—拉好」。
3. 每次只让孩子练其中一步。
4. 提前 10 分钟开始出门流程。

避免做法：
- 一边催一边批评。
- 每次都直接替孩子穿好。
- 在快迟到时训练孩子独立。

观察任务：
接下来 3 次出门，只练一个步骤，例如「自己把脚伸进去」。

---

## 4. 孩子吃饭乱扔

结论：
孩子扔食物可能是在探索因果、表达吃饱、觉得无聊，或餐桌边界不清楚。

蒙氏视角：
吃饭不是单纯摄入营养，也是在学习身体控制、餐桌秩序和边界。

家庭做法：
1. 食物少量摆放。
2. 平静说：「食物放在盘子里。」
3. 如果继续扔，平静结束这一轮。
4. 用适合孩子尺寸的餐具。

避免做法：
- 大声反应，让扔东西变成游戏。
- 追着喂。
- 用威胁或奖励交换吃饭。

观察任务：
观察孩子扔食物前，是不是已经吃饱、无聊或想离开餐椅。

---

## 5. 孩子总是打断大人

结论：
先判断孩子是有真实需求，还是还没有学会等待和轮流。

蒙氏视角：
等待、轮流、尊重他人工作，都需要通过环境和示范慢慢建立。

家庭做法：
1. 给孩子一个明确等待动作，例如把手放在成人手臂上。
2. 成人看到后回应：「我知道你有事，我说完这句就听你说。」
3. 真的说完后，立刻转向孩子。
4. 平时练习轮流说话。

避免做法：
- 完全无视孩子。
- 一打断就训斥。
- 成人自己也频繁打断孩子。

观察任务：
接下来 3 天，用固定等待动作替代喊叫打断。

---

## 6. 孩子反复撕纸

结论：
如果不是破坏重要物品，撕纸可能是手部精细动作和因果探索。

蒙氏视角：
孩子在感受手的力量、纸的质地、声音和变化。

家庭做法：
1. 准备可撕的废纸或旧杂志。
2. 给一个固定撕纸篮。
3. 示范撕完放进篮子。
4. 如果撕书，平静说：「书用来看，纸可以撕。」

避免做法：
- 一概禁止所有撕纸。
- 孩子撕书时大喊大叫。
- 不提供替代材料。

观察任务：
准备一个撕纸工作，看孩子是否会减少撕书行为。

---

## 7. 孩子不愿意分享

结论：
不要急着给孩子贴「自私」标签。年幼孩子需要先建立物品归属感，才可能真正理解分享。

蒙氏视角：
分享不是被迫交出手里的东西，而是在安全感和社交经验中逐渐发展出来的能力。

家庭做法：
1. 保护孩子正在使用的物品：「他用完会给你。」
2. 引导轮流，而不是强迫分享。
3. 准备可轮流的材料。
4. 成人示范如何等待。

避免做法：
- 「你怎么这么小气？」
- 强行从孩子手里拿走给别人。
- 用道德压力要求孩子分享。

观察任务：
观察孩子在不被强迫时，是否更愿意在结束后交给别人。

---

## 8. 孩子专注时被打断

结论：
孩子专注时，成人不要急着问、夸、拍照或展示。

蒙氏视角：
专注是儿童自我建构的重要状态。成人不必要的打断会破坏孩子的工作周期。

家庭做法：
1. 孩子投入时保持安静。
2. 不急着问「你在干什么」。
3. 不为了拍照打断动作。
4. 等孩子自然结束后再交流。

避免做法：
- 「真棒，再做一个给妈妈看。」
- 「你这个是什么颜色？」
- 频繁把孩子的工作变成表演。

观察任务：
下一次孩子专注时，成人不说话，记录他持续了多久。

---

## 9. 孩子睡前拖延

结论：
先看睡前流程是否稳定，环境是否过度刺激，而不是直接说孩子不听话。

蒙氏视角：
孩子需要稳定秩序来过渡到睡眠。睡前越混乱，孩子越难进入状态。

家庭做法：
1. 固定睡前流程：洗漱、绘本、关灯、拥抱。
2. 每晚顺序尽量一致。
3. 提前减少强刺激活动。
4. 用简短语言提示下一步。

避免做法：
- 每晚流程变化很大。
- 睡前讲太多道理。
- 一边催睡一边让孩子继续玩。

观察任务：
连续 5 天使用同一套睡前流程，看拖延是否减少。

---

## 10. 孩子怕脏

结论：
孩子怕脏可能和感官敏感、秩序感或成人反应有关，不要强迫孩子马上接受。

蒙氏视角：
感官体验需要循序渐进。孩子需要在可控、安全的环境里慢慢接触不同材料。

家庭做法：
1. 从干净、可控的材料开始，例如米、豆子、干沙。
2. 准备湿巾或洗手流程，让孩子知道可以清理。
3. 成人示范接触材料。
4. 不强迫孩子摸。

避免做法：
- 「这有什么好怕的？」
- 强行把孩子手按进材料里。
- 用嘲笑刺激孩子。

观察任务：
准备一个孩子可拒绝的感官材料，看他是否愿意靠近、观察或轻触。

---

## 11. 孩子总要自己来

结论：
「我自己来」通常是独立性发展的信号，不是故意和成人对着干。

蒙氏视角：
孩子在争取参与真实生活的能力。成人要保护这种独立冲动，同时设置安全边界。

家庭做法：
1. 允许孩子做安全部分。
2. 把任务拆小。
3. 成人只帮困难的最后一步。
4. 预留更多时间。

避免做法：
- 「你太慢了，我来。」
- 「你不会，别弄。」
- 所有事情都为了效率替代孩子。

观察任务：
今天选一件事，让孩子完成其中一个小步骤。

---

## 12. 孩子总是重复同一个游戏

结论：
重复不是浪费时间，可能是孩子正在通过重复掌握动作、顺序或概念。

蒙氏视角：
儿童通过重复形成专注和内在建构。成人不必急着增加难度。

家庭做法：
1. 允许孩子重复。
2. 观察重复的是动作、顺序还是结果。
3. 不急着教新玩法。
4. 等孩子自然失去兴趣后再替换材料。

避免做法：
- 「你怎么又玩这个？」
- 成人强行加入自己的玩法。
- 孩子刚进入重复就打断。

观察任务：
记录孩子重复了几次，以及每次动作是否更稳定。

---

## 13. 孩子拿东西乱倒乱翻

结论：
先看是不是环境过于开放，孩子还没有能力管理这么多物品。

蒙氏视角：
自由需要环境边界。东西太多、分类不清，孩子就会用翻倒来探索。

家庭做法：
1. 降低开放物品数量。
2. 使用托盘和篮子分类。
3. 每次只给一个工作。
4. 成人示范用完放回。

避免做法：
- 家里到处都是开放杂物。
- 孩子一翻就大骂。
- 没有固定收纳位置。

观察任务：
整理一个低刺激角落，看孩子是否减少乱翻。

---

## 14. 孩子不愿意刷牙

结论：
不要只靠命令，可以把刷牙变成孩子能参与、能选择、能预期的流程。

蒙氏视角：
刷牙是日常生活练习。孩子需要动作示范、有限选择和稳定流程。

家庭做法：
1. 固定刷牙时间和位置。
2. 给两个选择：「先刷上面还是下面？」
3. 成人慢速示范。
4. 必要时成人完成关键清洁，孩子完成参与部分。

避免做法：
- 问「你要不要刷牙？」
- 每次靠威胁完成。
- 让刷牙变成亲子权力斗争。

观察任务：
连续 5 天使用同一句提示和同一流程，看抵抗是否降低。

---

## 15. 孩子总说「不要」

结论：
「不要」可能是自主意识发展，不一定是故意反抗。

蒙氏视角：
孩子开始意识到自己可以选择。成人要把无限对抗改成有限选择。

家庭做法：
1. 少问开放式问题。
2. 给两个可接受选项。
3. 保持规则稳定。
4. 承认情绪，但不放弃边界。

例子：
「你不想现在穿鞋。我们要出门了，你想自己穿，还是我帮你穿第一只？」

避免做法：
- 和孩子争输赢。
- 用更多语言压制。
- 每次孩子说不要就完全改变规则。

观察任务：
今天把一个冲突问题改成两个选择，看孩子反应。
