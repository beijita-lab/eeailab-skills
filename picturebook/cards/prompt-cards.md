# 提示词卡

## 统一角色提示词模板

```text
主角是一个 {年龄} 岁左右的中国小孩，{发型}，{脸型和眼睛}，{常穿衣服}，表情自然可爱，动作真实，像普通家庭里的孩子，不要过度精致，不要成人化。
```

示例：

```text
主角是一个 2 岁左右的中国小女孩，圆脸，黑色短发，明亮眼睛，穿浅黄色家居服，表情自然可爱，动作真实，像普通家庭里的孩子，不要过度精致，不要成人化。
```

## 统一风格提示词模板

```text
{风格名称}，{材质 / 色彩 / 光线}，画面干净，背景简洁，适合亲子绘本，无文字，无水印，无 logo。
```

## 每页提示词模板

```text
画面：{这一页发生了什么}
角色：{统一角色描述}
动作：{主角正在做什么}
情绪：{主角表情和氛围}
场景：{地点和关键物品}
风格：{统一风格描述}
构图：{近景 / 中景 / 全景，横版 / 竖版 / 方形}
限制：无文字，无水印，无 logo，不要多余人物，保持角色一致。
```

## 负面提示词

```text
不要文字，不要水印，不要 logo，不要恐怖表情，不要成人化，不要过度写实，不要复杂背景，不要多余手指，不要畸形肢体，不要改变主角发型和服装，不要出现版权角色。
```

## 改图提示词

### 角色不一致

```text
请保持上一张图中主角的年龄、发型、脸型、服装和整体气质不变，只调整当前页面的动作和场景。
```

### 画面太复杂

```text
请简化背景，减少物品数量，只保留主角、关键道具和必要场景，让画面更适合低龄儿童绘本。
```

### 情绪不对

```text
请把主角表情调整为更自然、更温和的亲子绘本表情，不要夸张大笑或哭闹。
```

### 需要留白加文字

```text
请在画面下方或右侧保留干净留白区域，方便后期添加绘本文字。图片本身不要生成任何文字。
```

## 提示词输出原则

- 角色一致性比单张图惊艳更重要。
- 每页都要重复统一角色和统一风格。
- 不建议把中文标题或正文放入生图提示词。
- 如果用户要打印，提示词里要写清比例。
