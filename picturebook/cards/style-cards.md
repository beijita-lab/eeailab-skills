# 绘本风格卡

## 温暖水彩绘本风

适合：成长纪念、亲子陪伴、生活习惯。

关键词：

```text
温暖儿童绘本风，柔和水彩质感，低饱和度，干净背景，柔和自然光，细腻表情，适合 3-6 岁儿童亲子共读
```

## 扁平童趣插画风

适合：英语启蒙、动作指令、低龄认知。

关键词：

```text
简洁童趣插画，圆润造型，明亮色彩，干净背景，大块色彩，动作清晰，适合低龄儿童认知绘本
```

## 迪士尼感动画绘本风

适合：冒险、音乐、情绪丰富的故事。

关键词：

```text
高质量动画电影感儿童绘本，丰富表情，戏剧化光影，温暖色彩，故事感强，角色可爱但不夸张
```

注意：不要直接要求模仿具体版权角色。

## 纸质拼贴风

适合：手工感、家庭制作感、亲子活动。

关键词：

```text
纸质拼贴儿童绘本风，手工纹理，柔和阴影，简单形状，亲切自然，像父母和孩子一起做的手工书
```

## 中国家庭生活风

适合：真实家庭场景、行为引导。

关键词：

```text
中国普通家庭生活场景，温暖真实，家居环境简洁自然，孩子表情真实，亲子互动自然，不夸张，不商业广告风
```


## 温暖粘土动画风

适合：孩子和公仔 / 玩偶的晚安故事、玩具故事、睡前绘本。

关键词：

```text
温暖粘土动画风，手工黏土质感，圆润可爱的角色，柔和睡前灯光，低饱和暖色，梦幻但不夸张，像定格动画里的小世界，儿童绘本画面，干净背景，无文字，无水印，无 logo
```

注意：不要恐怖玩偶风，不要诡异表情，不要过度拟人，不要阴暗光影。

## 风格选择规则

- 行为引导：优先温暖水彩 / 中国家庭生活风。
- 英语启蒙：优先扁平童趣 / 动作清晰。
- 成长纪念：优先温暖水彩 / 纸质拼贴。
- 冒险故事：可以用动画电影感，但避免版权角色。
- 粘土娃娃晚安故事：优先温暖粘土动画风，避免恐怖玩偶感。
