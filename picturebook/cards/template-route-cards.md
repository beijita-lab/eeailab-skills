# 模板类 / 原创类分流卡

## 入口判断

用户说「想做绘本」时，先分流：

```text
模板类：适合快速做成品。
原创类：适合从 0 开始创作故事。
```

不要一上来直接生成故事，也不要一上来问完整 5 个细节。

## 模板类

适合用户：

- 想快速看到成品；
- 没有完整故事；
- 有孩子照片；
- 想做打印小书；
- 想做认知书；
- 想把孩子和公仔做成某种固定风格。

模板类可选：

```text
认知模板：身体、颜色、动作、生活物品、情绪、家庭成员。
故事模板：粘土娃娃晚安故事、生日成长书、刷牙睡觉、收玩具、第一次上幼儿园。
```

## 原创类

适合用户：

- 已经有故事；
- 想让 AI 帮忙创作新故事；
- 想做有起承转合的成长故事；
- 想做行为引导、情绪引导、英语启蒙故事或虚构冒险。

原创类必须追问：

```text
故事来源：用户提供 / AI 生成 / 共同改写
角色来源：用户提供 / AI 设计 / 孩子照片抽象
风格来源：用户指定 / AI 推荐 / 参考图
用途：打印 / 电子 / 视频 / 公众号
比例：打印前必须确认
```

## 推荐入口话术

```markdown
你可以先选一种路线：

1. **模板类**
   适合快速做成品。
   - 认知模板：身体、颜色、动作、生活物品、情绪、家庭成员；
   - 故事模板：粘土娃娃晚安故事、生日成长书、刷牙睡觉、收玩具、第一次上幼儿园。

2. **原创类**
   适合从 0 开始创作一个新故事，比如成长故事、行为引导、情绪引导、英语启蒙故事、虚构冒险故事。

如果你不确定，建议先选「模板类」，更容易做出来。

你想做模板类，还是原创类？
```
