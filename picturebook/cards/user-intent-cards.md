# 用户意图判断卡

## 0. 先判断路线：模板类 / 原创类

用户只说「我想做绘本」「想用孩子照片做一本书」「想做 AI 绘本」时，不要立刻追问完整细节，也不要直接创作故事。

先让用户二选一：

```text
模板类：认知书、粘土娃娃晚安故事、生日成长书、刷牙睡觉、收玩具等，适合快速做成品。
原创类：成长故事、行为引导、情绪引导、英语启蒙故事、虚构冒险等，适合从 0 创作情节。
```

### 模板类信号

- 用户有孩子照片、玩偶照片或固定素材；
- 用户想快速做出成品；
- 用户说「身体认知」「颜色认知」「动作认知」「认图」「认知书」；
- 用户说「粘土风」「公仔」「玩偶」「晚安故事」。

优先动作：先问具体模板，再问素材、用途、页数和打印比例。

### 原创类信号

- 用户想做一个有起承转合的故事；
- 用户说「成长」「情绪」「行为引导」「英语启蒙故事」「冒险」；
- 用户想让 AI 帮忙想故事。

优先动作：确认故事来源、角色来源、风格来源、用途和比例。

## 1. 成长纪念绘本

信号：

- 想用孩子形象做绘本；
- 记录孩子某个阶段；
- 想做生日礼物、成长礼物、家庭纪念。

优先问：

1. 孩子年龄和性格特点；
2. 想记录哪件真实小事；
3. 是写实温暖，还是童话化表达；
4. 是否要打印成册。

输出重点：真实感、家庭场景、温柔情绪、少说教。

## 2. 英语启蒙绘本

信号：

- 想做英文绘本；
- 想让孩子学句子、学单词；
- 想做中英双语。

优先问：

1. 孩子年龄；
2. 当前英语基础；
3. 想覆盖哪些词或句型；
4. 是否要配动作或亲子互动。

输出重点：重复句型、动作词、生活场景、短句，不做翻译灌输。

## 3. 行为引导绘本

信号：

- 不收玩具；
- 不刷牙；
- 喜欢扔东西；
- 睡前拖延；
- 分离焦虑；
- 不愿分享。

优先问：

1. 孩子年龄；
2. 具体发生场景；
3. 父母希望孩子理解什么；
4. 想温柔引导，还是幽默化处理。

输出重点：不贴标签，不说教，把问题变成孩子能理解的小故事。

## 4. 科普 / 兴趣绘本

信号：

- 孩子喜欢恐龙、车、动物、宇宙、植物；
- 想做知识型绘本；
- 想把兴趣变成阅读材料。

优先问：

1. 孩子喜欢的具体主题；
2. 年龄和理解能力；
3. 想科普多少知识；
4. 要故事化还是图鉴式。

输出重点：少量知识 + 具体画面 + 好奇心，不堆概念。

## 5. 已有故事改绘本

信号：

- 我已经有故事；
- 帮我拆成绘本；
- 帮我做分镜；
- 帮我生成每页提示词。

优先动作：

- 不重写故事，除非用户要求；
- 先拆页；
- 再给每页画面和提示词；
- 如故事过长，提示用户删减。

## 6. 用户只想要提示词

信号：

- 给我即梦提示词；
- 给我 Midjourney prompt；
- 我要拿去生图；
- 每页图片提示词。

优先动作：

- 先确认统一角色、统一风格、尺寸比例；
- 如果未确认，先给默认假设；
- 输出提示词包，不展开太多理论。
