# 孩子本人认知书卡

## 定位

孩子本人认知书是模板类绘本，不需要复杂故事。

核心结构：

```text
孩子本人照片 / 绘本化形象
+ 身体部位 / 颜色 / 动作 / 生活物品 / 情绪 / 家庭成员
+ 中英文认知词
+ 简单句子
+ 打印成册
```

## 适合年龄

```text
1-5 岁
```

## 默认页数

```text
10 页：封面 + 8 页正文 + 封底
```

## 身体认知模板

```text
封面：我的第一本身体认知书 / My First Body Book
第 1 页：Head 头
第 2 页：Eyes 眼睛
第 3 页：Nose 鼻子
第 4 页：Mouth 嘴巴
第 5 页：Hand 手
第 6 页：Foot 脚
第 7 页：Ear 耳朵
第 8 页：Hair 头发
封底：This is me. 这就是我。
```

## 每页版式

建议使用两块版式：

```text
左侧：孩子照片 / 绘本化形象 + 英文大词 + 中文词 + 简单句
右侧：局部特写 / 图标 / 认知对象 + 英文词 + 中文词
```

## 句式模板

### Eyes 眼睛

```text
Eyes 眼睛
My eyes help me see the world.
我的眼睛帮我看见世界。
```

### Hand 手

```text
Hand 手
My hands help me touch and hold.
我的手帮我触摸和拿东西。
```

### Foot 脚

```text
Foot 脚
My feet help me walk and run.
我的脚帮我走路和奔跑。
```

### Red 红色

```text
Red 红色
I see red.
我看见红色。
```

### Jump 跳

```text
Jump 跳
I can jump.
我会跳一跳。
```

## 照片素材要求

- 身体认知：准备正脸照、眼睛 / 手 / 脚等局部照片，或可从照片裁切。
- 颜色认知：准备孩子穿不同颜色衣服、拿不同颜色物品的照片。
- 动作认知：准备跑、跳、拍手、挥手、抱抱等动作照片。
- 生活物品：准备孩子和牙刷、杯子、鞋子、书、玩具等合照。

## 提醒

- 不建议直接让 AI 在图片里生成中文字。
- 文字后期排版更稳。
- 打印前先确认比例和尺寸。
