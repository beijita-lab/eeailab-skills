# 粘土娃娃晚安故事模板

## 模板定位

适合把孩子本人和他平时喜欢的公仔 / 玩偶一起做成一本温暖睡前绘本。

核心形式：

```text
孩子照片
+ 常玩的公仔 / 玩偶
+ 粘土动画风格
+ 晚安故事
+ 打印 / 电子 / 视频化
```

## 适合年龄

```text
2-6 岁
```

## 默认规格

```text
页数：10 页或 12 页
风格：温暖粘土动画风
情绪：睡前、温柔、梦幻、安全感
比例：打印前必须先向打印店 / 平台确认
```

## 用户需要提供

1. 孩子照片：正脸、自然表情，最好 3-5 张。
2. 公仔 / 玩偶照片：1-3 个，清楚拍出颜色和形状。
3. 孩子昵称。
4. 想要的故事场景：卧室、云朵小镇、玩具森林、月亮小屋等。
5. 用途：打印、电子阅读，还是视频。

## 默认故事结构：10 页

### 封面

- 标题：我的玩偶晚安绘本
- 画面：孩子和几个粘土风公仔坐在小床边，窗外有月亮。

### 第 1 页：夜晚到了

- 画面：房间灯光变柔和，孩子准备睡觉。
- 文案：夜晚到了，房间慢慢安静下来。

### 第 2 页：玩偶还醒着

- 画面：小熊、小兔或恐龙玩偶睁着眼睛。
- 文案：可是，玩偶们好像还没有睡着。

### 第 3 页：他们想听故事

- 画面：孩子抱着玩偶，玩偶围坐在一起。
- 文案：「可以讲一个晚安故事吗？」小熊轻轻问。

### 第 4 页：月亮小船来了

- 画面：窗边出现一艘小小的月亮船。
- 文案：一艘月亮小船，停在窗边等他们。

### 第 5 页：云朵小镇

- 画面：孩子和玩偶坐上月亮船，经过云朵小镇。
- 文案：他们飞过软软的云朵，看见一盏一盏小星灯。

### 第 6 页：星星送来晚安

- 画面：一颗小星星落在被子上。
- 文案：小星星说：「今天辛苦啦，晚安。」

### 第 7 页：玩偶们睡着了

- 画面：公仔们一个个躺好。
- 文案：小熊睡着了，小兔睡着了，恐龙也闭上了眼睛。

### 第 8 页：孩子也闭上眼睛

- 画面：孩子躺在床上，抱着最喜欢的玩偶。
- 文案：我也要睡啦，明天再一起玩。

### 封底

- 画面：月亮挂在窗外，房间安静温暖。
- 文案：晚安，亲爱的宝贝。

## 粘土风格提示词

```text
温暖粘土动画风，手工黏土质感，圆润可爱的角色，柔和睡前灯光，低饱和暖色，梦幻但不夸张，像定格动画里的小世界，儿童绘本画面，干净背景，无文字，无水印，无 logo。
```

## 孩子和玩偶一致性提示词

```text
把孩子和他的玩偶都转化为手工粘土角色。孩子保留原本的发型、脸型、服装特征和自然表情；玩偶保留原本颜色、形状和关键特征。整体像温暖定格动画里的粘土小世界。
```

## 注意

- 不要做成恐怖玩偶风。
- 不要让玩偶表情过度拟人或奇怪。
- 睡前故事要温柔，不要紧张冒险。
- 打印前先确认比例、尺寸、出血和页数。
